# 迭代1功能分解与GWT

## US-1 学生注册
1. 正常：用户名密码合法 → 注册成功，跳转到/login?registered
2. 异常：用户名已存在 → 注册失败，提示错误
3. 边界：密码为空 → 注册失败，提示密码不能为空

## US-2 用户登录
1. 正常：用户名密码正确 → 登录成功，跳转到/home
2. 异常：密码错误 → 提示用户名或密码错误
3. 边界：未登录访问保护页面 → 自动跳转到/login

## US-3 角色权限与菜单
1. 学生：只能访问/student/**，显示学生菜单
2. 教师：只能访问/teacher/**，显示教师菜单
3. 管理员：只能访问/admin/**，显示管理员菜单
4. 越权访问 → 返回403禁止访问
